revise-message extension demo

Overlay workflow for editing a previous assistant message, then either copying markdown to clipboard or sending it back to continue the conversation.

What this recording shows

  • Run /revise and pick a previous assistant message.
  • Edit inline markdown in the overlay editor.
  • Use Ctrl+Y to copy edited markdown to clipboard.
  • Run /revise again and use Ctrl+S to send edited markdown back to the assistant.
